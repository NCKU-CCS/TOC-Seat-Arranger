# TOC Seat Arranger

## Prerequisite
- Python3
- Excel

### Install Dependent Libraries
```sh
pip install -r requirements.txt
```

## Usage

```sh
python seat_arranger.py [-o OUTPUT_FILE] input_file
```

- input\_file
    - It should be a text file with list of student names

- OUTPUT\_FILE
    - Filename of output xlsx file
